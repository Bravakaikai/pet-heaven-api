# Pet Heaven Api (Game)

User can feed a pet and buy their foods in the shop.

* Frontend: Vue 3 (Composition API) > [GitHub link](https://github.com/Bravakaikai/pet-heaven-web)
* Backend: Kotlin 
* DataBase: PostgreSQL
